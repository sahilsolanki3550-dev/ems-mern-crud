# Employee Management System – MERN CRUD App

## Description
Employee Management System built using MERN stack with MongoDB.
Supports CRUD operations for managing employees.

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js

## Features
- Add employee
- View employees
- Update employee
- Delete employee

## Project Structure
/frontend → React application  
/backend → Node.js + Express + MongoDB API  

## How to Run Locally
### Backend
cd backend
npm install
npm start

### Frontend
cd frontend
npm install
npm start

## Note
MongoDB database is not included.
Use your own MongoDB URI in `.env` file.
